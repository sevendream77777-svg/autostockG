✔ 결론 1:
트레이닝 프로그램에서 -5일을 없애거나 조절하는 건 100% 가능

단,

“정답 없는 row를 학습에 쓰는 건 절대 불가능”

하지만
“그 row를 DB에 남긴다 → 예측용으로는 사용된다”
는 정상적으로 가능함.

2. 역으로 “종목을 지정해주면 예측 확률·수익 예측 가능?”
✔ 결론: 완전 가능

지금 구조에서 충분히 가능하고, 이미 구현된 기능과 매우 유사함.

예:

predict(stock="005930", date="2025-11-21")
→ 5일 뒤 수익률 예측
→ 확률 / 점수 / 리스크 / 순위까지 모두 가능


즉,
리얼엔진은 정답이 없는 미래 날짜에도 예측이 가능함.

왜?

예측(Prediction)은 Y(정답)가 필요 없다

X만 있으면 됨

미래 5일이 없어도 “예측값”은 나옴

✔ 이렇게도 가능
▣ A. 특정 종목만 예측

삼성전자 5일 뒤 수익률

카카오 1일 뒤 리스크

현대차 상승 확률

▣ B. 여러 종목 비교

S&P500 구성종목 중 상위 Top10

KOSPI200 종목 중 상승 확률 Top10

개인 보유 종목(포트폴리오) 자동 예측

▣ C. 외부 조건 필터

거래량 증가 종목 중 상위

PER 낮고 수급 좋은 종목 중 top

특정 업종만 예측

3. 결론 요약
✔ 트레이닝프로그램은

horizon(1d/2d/5d/10d)

input_window(20/60/120)

feature_group(HOJ/SLE 등)

train/valid split(9년/1년, 전체10년)
전부 파라미터로 통제 가능.

즉 “UI 기반 실험실” 100% 가능.

✔ 라벨 없는 마지막 5일

삭제하지 않고 Feature 파일처럼 남길 수 있음

학습에서는 제외됨

예측에서는 정상 사용됨

이것도 트레이닝 프로그램에서 제어 가능

✔ 종목 지정 예측

100% 가능

이미 HOJ/SLE 구조의 가장 중요한 기능

예측은 X만 필요(Y 필요 없음)

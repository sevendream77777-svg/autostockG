호슬(HOJ+SLE) 모델 개념 정리 (2025-11-19 기준)

1. 전체 목표

호엔진(HOJ)과 슬엔진(SLE)을 각각 독립적으로 최대 성능까지 끌어올린 뒤,
두 엔진의 예측 방식 차이를 이용해 조합·필터·병합·멀티엔진 전략으로
궁극의 수익률을 목표로 하는 통합 모델.

------------------------------------------------------------------------

2. 원본 데이터 구조

(A) HOJ 엔진 원본 데이터

-   개별종목 OHLCV (10년)
-   KOSPI 지수 (10년)
-   총 8개 원본칼럼

(B) SLE 엔진 원본 데이터

-   PER / PBR
-   기관 순매수
-   외국인 순매수
-   개인 매수/매도
-   총 4개 원본칼럼
    (샘플 약 50만개, 날짜별 단일 스냅샷 구조)

------------------------------------------------------------------------

3. 특징(Feature) 엔지니어링

HOJ 엔진

-   15개 기술적 지표 생성
    -   SMA, EMA, RSI, MACD, ATR, Stoch 등
-   KOSPI 지수와 결합
-   최종 약 15~40개 피처 확장 테스트 가능

SLE 엔진

-   네이버/한국투자증권 기반 PER/PBR/수급 데이터
-   “흐름 기반” 피처 및 “순매수비중/지속성” 피처 중심
-   샘플 기반: T일 하나 = 하나의 데이터포인트

------------------------------------------------------------------------

4. 학습 구조

HOJ 모델 (LightGBM Regressor)

-   입력: 60일 피처 시퀀스
-   출력: 5일 후 수익률
-   검증: 최근 1년 고정 Early-stopping

SLE 모델

-   입력: 단일 스냅샷 기반 피처(기관/외국인/PER/PBR)
-   출력: 5~60일 다중 타깃 성능 분석
-   검증: 구간별 IC, Top10 Return 평가

------------------------------------------------------------------------

5. 최종 엔진 조합 전략(궁극 목표)

1)  HOJ 단독 모델

-   기술적지표 중심 트렌드 감지에 뛰어남
    15일 피처 기본 , 40이상 피처 확장 트레인 가능

2)  SLE 단독 모델

-   실적/수급 중심 대형주 필터에 강함

3)  HOJ + SLE 합산 모델

-   두 엔진의 점수를 가중합

4)  HOJ 기본 + SLE 필터

-   HOJ가 추천한 종목 중
    SLE 상위 30~40%만 통과 → 압축형 강종목

5)  SLE 기본 + HOJ 필터

-   SLE가 추천한 종목 중
    HOJ 트렌드 필터로 날카롭게 선별

------------------------------------------------------------------------

6. 호슬모델의 궁극적 지향점

-   단기 트렌드(HOJ) + 중기/누적수급(SLE) 결합
-   머신러닝 2종류를 병렬·계층적으로 사용
-   시장상황 따라 두 엔진 비중 자동조절 가능

결론:
두 모델은 상호보완적이며,
‘기술적 흐름 + 펀더멘탈/수급’ 융합 모델이
장기적으로 가장 안정된 수익률을 가져온다.

------------------------------------------------------------------------

7. 향후 확장 계획

-   HOJ 15개 → 40개 피처 확대 실험
-   SLE 누락데이터 보강 후 완전체 DB 구축
-   HOJ/SLE 조합별 성능 백테스트 자동화
-   챔피언모델 자동선정 시스템 구축
